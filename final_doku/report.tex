\textbf{Abstract:}
This paper presents \app, a solution to control access to university buildings via a mobile device communicating with a NFC terminal at the door.
Heart of \app is an Android application, embedded in a holistic architecture with the goal of providing all services securely.
A NFC terminal enforces a secure NFC communication via an encrypted three-way handshake and handles physical door access.
A backend server provides authentication and basic authorization by managing a public key infrastructure.
For authenticity of users the backend interacts with TUMOnline, for authorization it communicates with the TUM Active Directory.
